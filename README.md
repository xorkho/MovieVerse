# MovieVerse

MovieVerse is a full-stack web application combining **Django backend** with a **React frontend**.  
It includes **RAG AI features** for movie data and interactive chat functionality.

## Features
- Browse and search movies from TMDB dataset
- Interactive chatbot using embeddings
- Modular frontend and backend architecture
- Easy to extend and deploy

## Tech Stack
- **Backend:** Python, Django, REST Framework
- **Frontend:** React.js, HTML, CSS, JavaScript
- **AI/ML:** Embeddings for retrieval-based chatbot
